# 魔法の遺産
『魔法の遺産』は『当方Project』にインスパイアされた弾幕シューティングゲーム。アーケードのコンティニュー方式を採用し、魔術士協会の審問官であるあなたは、王国で発生した奇妙な時代錯誤現象の原因を解明せねばならない。この異変の正体は何なのか？引き起こしているのは何か、あるいはだれか？様々な人々や生物の魔弾をかすめながら、この2.5D弾幕ゲームで自らその真相を突き止めよ！

本プロジェクトは、私一人で10週間かけて開発しました。音楽の一部は自分で作曲したが、それ以外の音楽やアート、グラフィック素材については、外部に委託したものやオンラインで入手したロイヤリティフリー素材を使用している。

# 操作方法（キーボード）

| ボタン | 機能 |
| --- | --- |
| Z | 弾発射／選択 |
| X | 大魔法 |
| シフト | フォーカス（低速移動） |
| 方向キー | 移動／UI操作 |
| Esc | ポーズメニュー表示 |

# コンパイル手順
自身でゲームをビルドまたは編集したい場合は、Unityバージョン2020.3.49.f1を使用して行うことができる。
