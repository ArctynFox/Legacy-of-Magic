# 魔法の遺産
『魔法の遺産』は『当方Project』にインスパイアされた弾幕シューティングゲーム。アーケードのコンティニュー方式を採用し、魔術士協会の審問官であるあなたは、王国で発生した奇妙な時代錯誤現象の原因を解明せねばならない。この再起の正体は？引き起こしているのは何か、あるいはだれか？様々な人々や生物の魔弾をかすめながら、この2.5D弾幕ゲームで自らその真相を突き止めよ！

このプロジェクトは、私一人だけで10週間かけて開発した。音楽の一部は自分で作曲したが、その他の音楽やアート、その他のグラフィック素材は、外部に委託したものやオンラインで入手したロイヤリティフリー素材を使用している。

[プレイ動画](https://www.youtube.com/watch?v=mVkjVgXQ0Bs)

# ドキュメント
ゲームの主要機能とデザイン思想に関するドキュメントは[こちら](/Documentation/Legacy%20of%20Magic%20-%20Design%20Document.pdf).

# ダウンロード
ゲームは[こちら](https://github.com/ArctynFox/Legacy-of-Magic/releases/latest)からダウンロードしてプレイできる.

# コンパイル手順
自身でゲームをビルドまたは編集したい場合は、Unityバージョン2020.3.49.f1を使用して行うことができる。
